# AWS Infrastructure Automation

Infrastructure as Code project demonstrating Terraform-based provisioning of AWS resources.

## Technologies

- Terraform
- AWS VPC
- EKS
- RDS
- IAM
- ALB
- Prometheus
- Grafana

## Features

- Modular Terraform configuration
- VPC provisioning
- EKS cluster setup
- RDS deployment
- IAM role management
- Application Load Balancer
- Monitoring stack configuration

## Commands

```bash
terraform init
terraform validate
terraform plan
terraform apply
```
